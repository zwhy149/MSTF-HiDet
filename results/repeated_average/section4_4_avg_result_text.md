In the revised reporting, the original AE7-copy Route B evaluation protocol was repeated across 5 random seeds (42, 52, 62, 72, 82) without altering the underlying method, feature set, losses, or the original code logic in the source folder. Under this repeated evaluation setting, MSTF-HiDet achieved an average L2 accuracy of 0.981 ± 0.014 and an average L2 macro-F1 of 0.983 ± 0.013. Accordingly, the previously reported single-run perfect result should be interpreted as one realization of the original protocol rather than the sole result used for manuscript-facing statistical reporting.
