# Split Files

This directory stores exported train/validation/test assignments derived from the AE9 Route B split logic.

## Files

- `route_b_seed42_split.csv`
  - the preserved original Route B split under seed 42

- `repeated_route_b_splits.csv`
  - split assignments for the 5 repeated seeds used in manuscript-facing average reporting

## Important Note

The public AE9 code still preserves the original split logic from the working codebase.
These exported CSV files are provided to make the evaluation partitioning more transparent for reviewers.
